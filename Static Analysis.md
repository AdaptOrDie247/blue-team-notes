# Linux Tools

## exiftool

Get file metadata
`exiftool filename`

## strings

Get strings from file
`strings filename`

## xorsearch

Search for encoded strings in a file (XOR, ROT, etc.)
`xorsearch filename searchstring`

Search for embedded executables
`xorsearch -p filename`
